# A Simple rest-api for a hairsalon :blue_heart:

This repo is deployed with heroku here:

```
https://simple-rest-api-for-hairsalon.herokuapp.com/
```

# To install

Clone this repo:

```
https://github.com/joakim-roos/hairsalon-rest-api.git
```

Then:

```
npm install
```

Then:

```
npm start
```

or

```
nodemon start
```
